## 🚀 Instalación Rápida
Para replicar este entorno en una máquina nueva con **EndeavourOS**:

1. **Clonar el repo:**
   ```bash
   git clone [https://github.com/Charlyp21/dotfiles.git](https://github.com/Charlyp21/dotfiles.git) ~/dotfiles
   cd ~/dotfiles
2. **Instalar paquetes base:**
   ```bash
   sudo pacman -S - < pkglist.txt
   yay -S - < aur-pkglist.txt
3. **Ejecutar el script de despliegue**
   ```bash
   chmod +x install.sh
   ./install.sh
